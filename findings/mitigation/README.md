Mitigation Files
================

These are example remediation files for the Part A transport/TLS issue:

- `MainActivity.java`: switches the `WebView` to HTTPS and cancels SSL errors.
- `network_security_config.xml`: disables cleartext traffic at the platform level.
- `AndroidManifest-snippet.xml`: shows the corresponding manifest change.

These are patch examples for the report and presentation. They are not a rebuilt APK.
